# 타이밍 정밀도 분석

## 트랜스폰더 → 베이스 지연 체인

```
자기장 감지 → LPCOMP 트리거    ~1us
MCU 웨이크업                   ~5us
HFINT 시작                     ~1us
Radio ramp-up                  ~40us
패킷 전송 (3B @ 2Mbps)         ~20us
RF 전파 (2~3m)                 ~10ns
베이스 수신 처리                ~10us
──────────────────────────────────
총 지연: ~77us
지연 변동 (jitter): ±5us
```

## 랩타임 정밀도 (단일 베이스)

```
고정 지연(~77us) 상쇄 → jitter만 영향
±5us = 28m/s 기준 ±0.14mm
```

## 가속 측정 정밀도 (듀얼 베이스)

```
Sync 패킷 수신 jitter: ±2us
TCXO 드리프트 (30초): ±15us
합계: ±17us = 28m/s 기준 ±0.5mm
```

극한 온도 환경에서 TCXO 특성이 다소 변할 수 있으나, ±0.1ms 이내 실용 정밀도 확보.

## 검출 신뢰도

```
와이어 유효 자기장 범위: ~10~20cm
28m/s 기준:
  10cm → ~3.5ms → ~40회 패킷 전송
  20cm → ~7ms  → ~80회 패킷 전송
40~80회 중 1회만 수신 성공하면 됨
```
