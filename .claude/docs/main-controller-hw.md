# 메인 컨트롤러 하드웨어 (가속 측정 모드 전용)

랩타임 모드에서는 불필요. 가속 측정 시에만 사용.

## 블록 다이어그램

```
[nRF52840 모듈]
      │
      ├── RADIO: ESB TX (sync 비콘) / ESB RX (릴레이 수신)
      │          채널 40, 주소 0xA1A1A1A1A1
      │          TX 파워: +8dBm
      │
      └── USB → [타이밍 PC]
```

별도 TCXO 불필요 (자체 타이밍이 아니라 베이스들의 타임스탬프를 수집/계산만 함).

## 프로토타입

nRF52840-DK 1개 + USB → PC. 추가 회로 없음.
