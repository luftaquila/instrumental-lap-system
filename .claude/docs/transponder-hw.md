# 트랜스폰더 하드웨어

## 블록 다이어그램

```
[CR2032] ─→ [nRF52840 모듈]
                  │
                  ├── AIN ←── 엔벨로프 검출 ←── LC 공진 코일
                  │           (다이오드 + RC)     (페라이트 코어 + 튜닝 캡)
                  ├── LPCOMP ← 엔벨로프 DC 레벨 감시 (슬립 중 웨이크업)
                  ├── RADIO → 2.4GHz TX (Enhanced ShockBurst)
                  ├── GPIO IN ← 홀 센서 (전원 ON/OFF)
                  └── GPIO OUT → LED (상태 표시)
```

## 주요 부품

| 부품 | 사양 | 역할 |
|------|------|------|
| nRF52840 모듈 | E73-2G4M08S1C 등 | MCU + 2.4GHz Radio |
| 페라이트 코어 코일 | 인덕턴스 TBD (125kHz 튜닝) | 자기장 감지 |
| 튜닝 캡 | C = 1/(4pi^2 f^2 L) | LC 공진 주파수 맞춤 |
| 검파 다이오드 | BAT54 또는 1N5711 | 엔벨로프 검출 |
| 홀 센서 | DRV5032 (SOT-23) | 외부 자석으로 전원 ON/OFF |
| LED | 0402 또는 0603 | 상태 표시 (포팅 표면으로 투과) |
| CR2032 홀더 | SMD | 전원 |

## LC 코일 튜닝

```
공진 주파수 f = 125kHz
코일 인덕턴스 L을 측정 후 C 계산:
  C = 1 / (4pi^2 * f^2 * L)

예: L = 10mH → C = 162pF
예: L = 4.7mH → C = 345pF
```

Q 팩터 목표: Q ~ 20~50. 높을수록 감도 향상, 대역폭 축소. 필요 시 병렬 저항으로 Q 조정.

## 웨이크업 회로

```
LC 코일 출력
  → 검파 다이오드 (반파 정류)
  → RC 평활 (tau ~ 100us)
  → nRF52840 LPCOMP (내장 저전력 비교기)
  → 임계 초과 시 인터럽트 → MCU 웨이크업
```

LPCOMP는 슬립 중에도 ~1uA로 동작. 외부 비교기 불필요.

## 동작 시퀀스

```
[슬립] ─── LPCOMP 트리거 ───→ [웨이크업]
  ↑                              │
  │   ┌──────────────────────────┘
  │   ├─ HFINT 시작              (~1us)
  │   ├─ RADIO 설정              (~5us)
  │   ├─ TX ramp-up              (~40us)
  │   ├─ 패킷 송출               (~20us)
  │   ├─ 반복 (LPCOMP 활성 동안)
  │   │    ~80us 간격 연속 전송
  │   │    검출 영역 ~10cm → ~3.5ms 동안 ~40회
  │   └─ LPCOMP 해제 → TX 중단
  └──── [슬립 복귀]
```

## 전력 추정

| 상태 | 전류 | 비고 |
|------|------|------|
| Deep sleep + LPCOMP | ~3uA | 99.99% 시간 |
| TX burst (-20dBm) | ~5mA | ~3.5ms per crossing |

```
랩타임 60초 기준:
I_avg = 3uA + (5mA * 3.5ms / 60s) = 3uA + 0.29uA ~ 3.3uA
CR2032 (220mAh) → 220,000uAh / 3.3uA ~ 66,000시간
```

자연 방전(연 ~1%)이 수명 제한 요인. 100시간 요구사항 대비 수천 배 마진.

## 외장

```
  ┌─────────────────────────┐
  │  ┌───┐ ┌─────────────┐  │
  │  │LED│ │ nRF52 모듈   │  │
  │  └───┘ └─────────────┘  │
  │  ┌────┐ ┌────────────┐  │  ~40 x 30 x 15mm
  │  │Hall│ │ CR2032      │  │  우레탄 포팅
  │  └────┘ └────────────┘  │
  │  ┌──────────────────┐   │
  │  │ 페라이트 코일      │   │
  │  └──────────────────┘   │
  └─────────────────────────┘
```

- PCB + 부품 + 코인셀을 실리콘 몰드에 배치 후 우레탄 포팅제(Polytek Poly 74-20)로 충전 경화
- 물리적 개구부 없음 → IP68 방수
- LED는 포팅 표면 가까이 배치 → 빛 투과
- 홀 센서도 표면 가까이 → 외부 자석 터치로 전원 ON/OFF
- 롤후프 파이프에 고무줄로 고정
